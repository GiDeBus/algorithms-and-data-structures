
## Problems

Given two strings, write a function to determine if the second string
is an anagram of the first. 

Note: An anagram os a word, phrase, or name formed by rearranging
the letters of another, such as cinema, from iceman.

Write a function called same, which accepts two arrays.
The function should return true if every value in the array has it's
corresponding value squared in the second array. The frequency of 
values must be the same.
